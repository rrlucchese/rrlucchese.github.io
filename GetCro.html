Command GetCro

Comand format

GetCro File-1 File-2 ...

This command computes the total cross section and photoelectron asymmetry parameters from the dynamical coefficients.

If the variables File-n are present, then the dynamical coefficients in those files are processed. Otherwise, the program use the automatically generated scratch file as described in the definition of the command PhIon. If the PlotData file has been defined then the data is also written to that file.

Data records used

  • IPot, the ionization potential in eV.

Optional data records used

  • Label, used in the text written to PlotData.
  • PolSel, Used to indicate which polarization direction of linrarly polarized light to include in the photoionization calculation, 0 for all directions (default), 1 for x, 2 for y, and 3 for z.
  • RotateFormIdy, Formula for rotation of photoionization matrix elements before computing cross section. Yields the same results as using the corresponding RotateForm before computing the cross sections.
  • CroByPartialWave, If this resord is present then the cross section is computed as a function of l.
