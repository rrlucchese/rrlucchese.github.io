Command DumpMesa

Command format

DumpMesa File(s)

This command writes out the geometry, basis set, and orbitals to file File in a form of input to the Mesa Quantum Chemsitry. If more than one file is present then separate data and put onto different files using the order, Geometry, Bound State Basis, Orbital Expansion Coeficients, Geometry with repeated centers for expanded basis definition

Optional data records used

  • AddCenterMesa, gives the location of an additional center for DumpMesa
