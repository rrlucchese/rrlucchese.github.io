Sample Jobs

Below is a list of the test scripts that are found in the directory ePolyScatE/tests. Required environment variables are discussed here.

Test Scripts

Almost of all of the tests have four files associated with them, the .inp file contains the input data, the .ostnd file which is the standard output of this job run, the output from a standard quantum chemistry code which is used as input by the scattering codes (eg .g03 or .molden2012), and the job used to run the quantum chemistry code (eg .g03.inp or .molpro.inp) Note that there are a changes in the format of the molden files produced by Molpro depending on the version.
