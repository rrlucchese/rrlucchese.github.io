Command ReadBlms

Command format

ReadBlms File

Where File is file where the data is to be read from. This data is created using using the SaveBlms command. Note that in addition to the Blms, the information that is obtained from the Convert command is read in, so that the geometry, basis set, and molecular orbital expansion coefficients are also read in. Thus there is no need to run the Convert command. This was done since the Blms depend on geometry for high partial wave and this guarantees that the Blms and geometry are consistent.
