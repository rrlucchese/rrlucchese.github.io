Command GeomNormMode

Command format

GeomNormMode iMode,xMode1 xMode2 ...

or

GeomNormModeN NMode iMode(1:NMode) xMode1(1:NMode) xMode2(1:NMode) ...

Used to generate geometries at different values of the normal coordiantes as determined using SymNormMode. iMode indicates the normal mode and xModei indicates the extension in units of the displacement at the classical turning point in the ground vibrational state. If defined, NMode indicates the number of modes which are simultaneously displaced.

Optional data records used

  • AltEqGeom, an alternate equilibrium geometry used to generate the geometries
