Environment variables required

TMPDIR
scratch directory on the master node, on some systems this must be visible to all nodes for the wrapper to work correctly
COMPILER
which compiler is used
  • gfortran (on MacOS, openmpi and Accelerate system library)
  • intelmkl (intel compilers, mkl libraries)
  • intel (on cori.nersc.gov using the ftn compiler wrapper in the PrgEnv-Intel environment)
MACH
Machine type, defined values are:
  • apple (Apple macOS 10.12 Sierra)
  • apple_10.14_rrl (Apple macOS Mojave)
  • cori (cori.nersc.gov - using ftn compiler, NERSC)
  • lrc_rrl and lrc (lrc-login.lbl.gov, Lawrencium Cluster)
pe
root directory for ePolyScat (e.g. ~lucchese/Applications/ePolyScat )
