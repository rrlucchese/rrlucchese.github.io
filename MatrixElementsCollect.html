Command MatrixElementsCollect and MatrixElementsCombine

Comand format

MatrixElementsCollect File-1 File-2 ...

MatrixElementsCombine File-1 File-2 ...

The command MatrixElementsCollect reads T-matrix elements from an external file and puts them into an internal data structure. The command MatrixElementsCombine reads T-matrix elements from an external file and combines them with matrix elements that have already read in for the same symmetry and energies. This is used to replace the low partial wave matrix elements with more accurate values. If the variables File-n are present, then the T-matrix elements in those files are processed. Otherwise, the program uses the automatically generated scratch file as described in the definition of the command Scat.
