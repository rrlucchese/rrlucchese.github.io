Command TotalCrossSection

Comand format

TotalCrossSection

This command computes the total and partial cross sections and eigenphase sums for electron scattering using the internal data structure for the T-matrix elements. If the internal data structure is not present, then the T-matrix elements are taken from the scratch file used for the output from Scat. Note that the internal data structure can be created from previously computed matrix elements stored in an external file by using the MatrixElementsCollect and MatrixElementsCombine commands. If the PlotData file has been defined then the data is also written to that file. If LMaxK is less than the value used to compute the scattering matrices, then the scattering matrices will be truncated down to LMaxK in the eigenphase sum and cross section caluclations

Optional data records used

  • Label, used it the text written to PlotData.
  • LogLogInterp, flag to indicate if the missing cross sections should obtained using log-log interpolation.
