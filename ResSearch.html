Command ResSearch

Comand format

ResSearch

This command searches for resonances using the adiabatic potential computed using GetDPot command. If the command FileName has been used to define the PlotData file, then the positions of the poles of the S matrix are written there.

Data records used

  • ResSearchEng, defines the region of the search in the complex plane.

Optional data records used

  • Label, used it the text written to PlotData.
  • RungeKuttaFac this integer controls the step size used in the differential equation integrator. Higher values yields smaller step sizes. Default value is 4.
  • ResSearchType This integer controls how the resonance search is performed. Default value is 0.
