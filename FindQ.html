Command FindQ

Command format

FindQ (Center Number(i), Atomic Number(i) Atomic Type(i), X(i), Y(i), Z(i), i = 1, Number of Centers)

Determines the values of the normal coordinates as determined using SymNormMode. for the specified geometry. The geometry is given as input to the command in the format of the "Standard Orientation" as printed by Gaussian.

Optional data records used

  • AltEqGeom, an alternate equilibrium geometry used to analyze the input geometries
