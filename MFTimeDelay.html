Command MFTimeDelay

Comand format

MFTimeDelay File-1 File-2 ...

or

MFTimeDelay Energy File-1 File-2 ...

This command computes the molecular frame amplitudes and the photoionization time delay from the energy derivative of the phases.

if the variable Energy is present then the results are written for only one photoelectron kinetic energy. If the variables File-n are present, then the dynamical coefficients in those files are processed. Otherwise, the program uses the automatically generated scratch file as described in the definition of the command PhIon.

Data that are just a function of the energy are written to PlotData if more than one energy is computed as defined in the FileName command, and data of two variables, e.g. energy and one angle, are written to PlotData2D.

Data records used

  • IPot, the ionization potential in eV.
  • MFTimeDelayAngles, gives the angles over which the time delays are to be computed and averaged

Optional data records used

  • Label, used in the text written to PlotData or PlotData2D
  • TargCompSel, for a degenerate target state, do only one of the components. The default average over all components.
  • TestOut, if present then annotate important output using "Test2Re" at the begining of the important results.
  • RotateFormIdy, Formula for rotation of photoionization matrix elements before computing cross section. Yields the same results as using the corresponding RotateForm before computing the cross sections.
