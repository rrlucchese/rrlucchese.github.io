Command SymNormMode

Command format

SymNormMode

The is used to computed the symmetry adapted normal modes that correspond to the symmetries obtained using GetBlms. It assumes that in the Quantum Chemistry output the normal modes were computed. For Gaussian that is done using the "Freq=HPModes" option.

Optional data records used

  • FreqToler, tolernace use to determine if vibrational frequency eigenvalues are degenerate (in units of cm-1). Default value is 1.0e-4.
